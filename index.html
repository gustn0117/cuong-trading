[ 계정 탈퇴 안내 및 주의사항 ]

탈퇴 전 반드시 확인해 주세요!

  • 영구 삭제 및 복구 불가: 계정 탈퇴 시 모든 이용 기록과 데이터는 즉시 삭제되며, 어떠한 경우에도 복구할 수 없습니다.
  • 재가입 제한: 탈퇴 완료 후 동일한 휴대폰 번호, 이메일, ID를 사용해 다시 가입하시려면 30일의 대기 기간이 필요합니다.
  • 자산 및 포지션 정리: 탈퇴 진행 전, 현재 보유 중인 모든 포지션을 종료(청산/매도)하고 계좌 잔액을 전액 출금해 주세요. 진행 중인 출금 요청이 있다면 완료된 후 탈퇴가 가능합니다.

모바일 앱에서 계정 탈퇴하기

  1. 프로필 설정 진입: 홈 화면 좌측 상단의 [메뉴] 아이콘 클릭 후, 상단 [프로필 바]를 선택합니다.
  2. 탈퇴 메뉴 선택: 하단의 [계정 삭제] 버튼을 누릅니다.
  3. 자산 확인: Web3 지갑을 포함해 남은 자산이 없음을 확인하는 체크박스에 표시하고 진행합니다.
  4. 최종 확인 및 인증: 위험 고지 사항을 숙지하신 후, 계정 비밀번호 입력 및 보안 인증을 완료해 주세요.
  5. 검토 및 완료: 탈퇴 요청은 24시간 이내에 검토되며, 이후 계정은 영구적으로 비활성화됩니다.

중요 참고 사항

탈퇴가 완료되면 모든 API 키와 연동 기기가 해제되며, 기존에 받으셨던 보너스나 혜택은 모두 소멸됩니다. 특히 은행 송금 기능을 이용하셨던 경우, 추후 새 계정에서 동일한 신분증으로 해당 기능을 재사용하는 것이 제한될 수 있으니 신중히 결정해 주시기 바랍니다.
